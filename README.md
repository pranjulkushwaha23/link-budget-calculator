# RF Link Budget Calculator

MATLAB script for calculating link budget of drone 
communication systems.

## System Parameters
- Frequency: 865–867 MHz ISM band
- TX Power: 30 dBm (1W)
- Achieved range: 15 km LOS
- Link margin: >47 dB

## Usage
Run `link_budget.m` in MATLAB. Adjust parameters 
in the config section at the top.

## Applications
Drone telemetry, long-range RF communication, 
SiK protocol systems.
